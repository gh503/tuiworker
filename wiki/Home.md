# Home

> TUIWorker - 一个基于 Rust 的终端用户界面（TUI）工作站应用
> TUIWorker - A Rust-based Terminal User Interface Workstation Application

## 欢迎 / Welcome

TUIWorker 是一个纯 Rust 编写的终端工作站应用，为终端爱好者提供统一、高效、可定制的命令行工作环境。

TUIWorker is a pure Rust-based terminal workstation application, providing a unified, efficient, and customizable command-line working environment for terminal enthusiasts.
