# 快捷键 / Keyboard Shortcuts

## 中文 / Chinese

### 全局快捷键 / Global Shortcuts

| 快捷键 | 功能 |
|--------|------|
| `q` | 退出应用 |
| `Ctrl + Tab` | 切换到下一个模块 |
| `Ctrl + Shift + Tab` | 切换到上一个模块 |
| `?` | 显示帮助 |
| `Ctrl + L` | 开关日志面板 |
| `Ctrl + ↑` | 增大日志面板高度 |
| `Ctrl + ↓` | 减小日志面板高度 |

### FileBrowser 模块快捷键 / FileBrowser Module Shortcuts

| 快捷键 | 功能 |
|--------|------|
| `j` / `k` 或 `↑` / `↓` | 上/下移动 |
| `Enter` | 进入目录 |
| `o` | 打开文件 |
| `u` / `Backspace` | 返回上级目录 |
| `h` | 显示/隐藏隐藏文件 |
| `s` | 切换排序方式 (名称/大小) |
| `m` | 按修改时间排序 |
| `d` | 删除文件 (按 d 弹出确认，再按 y 确认) |
| `r` | 重命名文件 (输入新名称后按 Enter 确认) |
| `/` | 搜索文件 |
| `Tab` | 在文件列表和内容区间切换 |
| `p` | 复制文件/文件夹路径 |
| `y` | 复制选中文本 |
| `Esc` | 关闭文件/退出搜索/取消删除 |
| `Enter` | 进入目录/打开文件 |
| `Backspace` | 返回上级目录 |
| `PageUp` / `PageDown` | 翻页 |
| `Home` / `End` | 跳到首/尾 |
| `←` / `→` | 左右滚动内容 |
| **鼠标拖拽** | 选择文本 |
| **右键点击** | 复制选中文本到剪贴板 |
| **Ctrl + C** | 复制选中文本 |

## English

### Global Shortcuts

| Shortcut | Function |
|----------|----------|
| `q` | Quit application |
| `Ctrl + Tab` | Switch to next module |
| `Ctrl + Shift + Tab` | Switch to previous module |
| `?` | Show help |
| `Ctrl + L` | Toggle log panel |
| `Ctrl + ↑` | Increase log panel height |
| `Ctrl + ↓` | Decrease log panel height |

### FileBrowser Module Shortcuts

| Shortcut | Function |
|----------|----------|
| `j` / `k` or `↑` / `↓` | Move up/down |
| `Enter` | Enter directory |
| `o` | Open file |
| `u` / `Backspace` | Go to parent directory |
| `h` | Toggle show hidden files |
| `s` | Toggle sort by (name/size) |
| `m` | Sort by modified time |
| `d` | Delete file (Press d to confirm dialog, then y to delete) |
| `r` | Rename file (Input new name, then Enter to confirm) |
| `/` | Search files |
| `Tab` | Switch between file list and content |
| `p` | Copy file/folder path |
| `y` | Copy selected text |
| `Esc` | Close file/exit search/cancel delete |
| `Enter` | Enter directory/open file |
| `Backspace` | Go to parent directory |
| `PageUp` / `PageDown` | Page up/down |
| `Home` / `End` | Go to start/end |
| `←` / `→` | Scroll content left/right |
| **Mouse Drag** | Select text |
| **Right Click** | Copy selected text to clipboard |
| **Ctrl + C** | Copy selected text |
