# 功能特性 / Features

## 中文 / Chinese

### 核心功能 / Core Features

| 功能 | 描述 | 状态 |
|------|------|------|
| 📁 **文件管理器** | 浏览文件系统，文本预览，打开外部文件 | ✅ 可用 |
| ✅ **待办事项** | 完整的 TODO 管理，支持标签、优先级 | 🏗️ 开发中 |
| 📝 **笔记** | Markdown 编辑，标签系统，快速记录 | 🏗️ 开发中 |
| 📅 **日记** | 日历视图，农历显示，节假日标记 | 🏗️ 开发中 |
| 🎵 **音乐播放** | 支持 FLAC/MP3/AAC 等无损格式 | 🏗️ 开发中 |
| 📮 **邮件收发** | IMAP/SMTP 客户端，纯 Rust 实现 | 🏗️ 开发中 |
| 📊 **项目跟踪** | 里程碑、风险、进度管理 | 🏗️ 开发中 |
| 💻 **终端模拟** | 嵌入式终端，多标签，分割窗口 | 🏗️ 开发中 |
| 🔀 **Git 操作** | 仓库状态、提交、日志查看 | 🏗️ 开发中 |

### 技术特性 / Technical Features

- **纯 Rust 实现** - 不依赖外部命令行工具
- **模块化架构** - 插件式设计，按需启用功能模块
- **高性能** - 基于 ratatui 和 tokio，流畅的终端交互体验
- **可定制** - 支持主题、快捷键、模块配置的完全自定义
- **跨平台** - 支持 Linux、macOS 和 Windows
- **本地优先** - 所有数据存储在本地，保护隐私

## English

### Core Features

| Feature | Description | Status |
|---------|-------------|--------|
| 📁 **File Browser** | Browse filesystem, preview text, open external files | ✅ Available |
| ✅ **Todo List** | Complete TODO management with tags and priorities | 🏗️ In Development |
| 📝 **Notes** | Markdown editing, tag system, quick note-taking | 🏗️ In Development |

### Technical Features

- **Pure Rust Implementation** - No external CLI tools required
- **Modular Architecture** - Plugin-style design, enable features as needed
- **High Performance** - Built on ratatui and tokio for smooth terminal experience
- **Highly Customizable** - Full customization of themes, shortcuts, and module configurations
- **Cross-Platform** - Supports Linux, macOS, and Windows
- **Local-First** - All data stored locally for privacy protection
