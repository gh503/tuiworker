# TUI Worker - 办公工作入口

一个基于 Rust 和 Ratatui 的跨平台终端用户界面（TUI）应用程序，作为 PC 办公工作入口。主要目标是提供纯命令行操作体验，类似于 vim 但功能更强大。

## 功能特性

- 🚀 **跨平台支持** - Windows、Linux、macOS
- ⚡ **高性能** - 120FPS 渲染，轻量级依赖
- 📝 **笔记管理** - 快速记录、分类、标签、搜索
- ✅ **待办管理** - 状态跟踪、优先级、完成时间
- 💻 **命令执行器** - 系统命令执行、历史记录
- 📁 **文件浏览器** - 快速导航和文件浏览
- 📅 **日历视图** - 查看日期、日程安排
- 🔍 **全局搜索** - 跨模块搜索笔记和待办事项
- 💾 **数据持久化** - JSON 格式存储，自动保存

## 安装

确保已安装 Rust（1.70+）：

```bash
# 克隆或创建项目
git clone https://github.com/yourusername/tuiworker.git
cd tuiworker

# 构建
cargo build --release

# 运行
cargo run --release
```

## 快速开始

### 基本操作

| 按键 | 功能 |
|------|------|
| `q` | 退出程序 |
| `Tab` / `Shift+Tab` | 切换标签页 |
| `1-8` | 直接跳转到标签页 |
| `↑` / `↓` | 上下导航 |
| `←` / `→` | 左右导航（文件浏览器） |
| `Enter` | 确认/执行 |
| `Esc` | 取消/退出模式 |

### 标签页

#### 1. 仪表板 (Dashboard)
显示欢迎信息、快捷键说明和统计数据。

#### 2. 笔记 (Notes)
- 快捷键：
  - `a` - 添加新笔记（在模态对话框中输入标题）
  - `/` - 过滤笔记
- 功能：
  - 创建、查看笔记
  - 按标题/内容搜索
  - 分类和标签支持

#### 3. 待办 (Todos)
- 快捷键：
  - `a` - 添加新待办事项
  - `Enter` - 切换状态
  - `n` - 显示全部
  - `f` - 只显示已完成
- 功能：
  - 状态管理（待办/进行中/已完成/已取消）
  - 按状态过滤
  - 优先级设置

#### 4. 命令 (Commands)
- 快捷键：
  - 输入命令后按 `Enter` 执行
- 功能：
  - 执行系统命令
  - 查看历史记录（最近100条）

#### 5. 日历 (Calendar)
- 功能：
  - 查看当前月份
  - 高亮显示今天
  - 显示事件标记

#### 6. 文件 (File Browser)
- 快捷键：
  - `→` 或 `Enter` - 进入目录
  - `←` - 返回上级目录
- 功能：
  - 浏览文件系统
  - 目录优先排序
  - 文件大小显示

#### 7. 搜索 (Search)
- 功能：
  - 全局搜索笔记和待办事项
  - 按标题/内容匹配

#### 8. 设置 (Settings)
- 功能：
  - 查看统计数据
  - 配置选项（开发中）

## 数据存储

数据默认存储在：
- Windows: `%APPDATA%\tuiworker\data.json`
- Linux: `~/.config/tuiworker/data.json`
- macOS: `~/Library/Application Support/tuiworker/data.json`

自动备份功能会在每次保存时创建备份文件（`.json.bak`）。

## 技术栈

- **Rust** - 系统编程语言
- **ratatui** (0.29) - 现代 TUI 框架
- **crossterm** (0.28) - 跨平台终端操作
- **serde** (1.0) - 序列化/反序列化
- **chrono** (0.4) - 日期时间处理
- **uuid** (1.6) - 唯一标识符生成
- **dirs** (5.0) - 平台特定路径

## 项目结构

```
src/
├── main.rs      # 应用入口和主循环
├── app.rs       # 应用状态和逻辑
├── models.rs    # 数据模型
├── event.rs     # 事件处理
├── ui.rs        # UI 渲染
├── storage.rs   # 数据持久化
└── mod.rs       # 模块声明
```
